#  Eco E-commerce Website

An eco-friendly e-commerce website built using **HTML, CSS, and JavaScript**.  
This project includes **live product search**, **dark mode**, and a clean, responsive UI.

##  Features

-  **Live Search Functionality**
  - Products filter as you type
  - Exact match on Enter key
- **Dark Mode**
  - Yellow-themed scrollbar
  - Yellow input text in dark mode
- **Product Listing UI**
  - Category-based products
  - Add to Cart & Buy Now buttons
-  **LocalStorage**
  - Stores products and login state
-  **Responsive Design**
  - Works on desktop and mobile

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Font Awesome  
- LocalStorage
